[ 웹사이트 제작 팀 과제 ]

1. 팀 구성

   별도 팀 구성 문서 참고
   
   팀 구성 문서에서 각 팀의 첫 번째 팀원이 임시 팀 리더 역할을 수행
   (팀 미팅을 통해 변경 가능)
   
   팀 미팅은 zoom 회의와 같이 비대면 방식으로 진행
   
2. 기간

   2021.09.07 ~ 2021.09.28 
   
   2021.09.28 15:00 프로젝트 발표
   
3. 작업 내용

   가. 팀 별로 프로젝트 주제 선정

        주제는 자유롭게 결정하되 데이터 수집, 처리, 표현을 포함하도록 선정

   나. 다음과 같은 기능을 포함해야 합니다.

        . 데이터 수집 ( 수집 산출물은 csv 파일 형태 )

        . 수집된 데이터를 데이터베이스에 저장

        . 데이터베이스의 데이터를 조회해서 화면에 표와 그래프 형태로 표시

        . 회원가입, 로그인, 로그아웃, 마이페이지에서 정보 수정 기능

        . 사용자 관리 기능 ( 사용자 목록 표시, 사용자 상세 정보 표시, 사용자 정보 수정, 권한 부여 등 기능 ). 단, 관리자만 수행 가능하도록 구현

        . 표시된 데이터에 대한 코멘트 등록, 수정, 삭제 기능 구현

        . 게시판 기능 ( 글쓰기, 목록보기, 수정, 삭제, 댓글쓰기 등 )

        . 그 외 팀별로 원하는 기능
