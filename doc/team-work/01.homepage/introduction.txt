[ 홈페이지 제작 팀 과제 ]

1. 팀 구성

   별도 팀 구성 문서 참고
   
   팀 구성 문서에서 각 팀의 첫 번째 팀원이 임시 팀 리더 역할을 수행
   (팀 미팅을 통해 변경 가능)
   
   팀 미팅은 zoom 회의와 같이 비대면 방식으로 진행
   
2. 기간

   2021.08.18 ~ 2021.08.25 
   
   2021.08.25 마지막 시간에 산출물 공개 
   
3. 진행 과정

   가. 팀 별로 홈페이지 주제 선정 (예: 영화정보, 게임정보, 도서정보, .... )

   나. 주제와 관련해서 역할 부여

      각 팀원은 1~2개의 페이지를 작성 하도록 역할 부여
   
      팀원 중 1인은 시작페이지(메인페이지) 작성

   다. 홈페이지 구성

      각 팀원이 제작한 개별 페이지를 통합해서 하나의 홈페이지 구현

      시작페이지(메인페이지)에서 각 팀원이 구성한 페이지로 연결되는 링크 구현
   
      필요한 경우 각 페이지는 링크를 통해 서로 연결되도록 구현

      visual studio code로 페이지 개발 ( 홈페이지 저작도구와 같은 자동화 도구 사용 배제 )
